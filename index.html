CSS Selectors

Selector Type HTML Element CSS Selector
Type Selector aka Tag Name or Element Selector <h1>Text</h1> p {
color: red;
}
Class Selector <h1 class='brand'>Text</h1> .brand {
color: red;
}
ID Selector <h1 id='large-title'></h1> #h1 {
color: red;
}
Attribute Selector <a href=''></a> [href] {
color: red;
}
Attribute Selector with Type and Value <img src='/images/winter.jpg'> img[src*='winter'] {
color: red;
}
Pseudo-Class Selector <h1>Text</h1> h1:hover {
color: red;
}
Multiple Classes <h1 class='green bold'>Text</h1> .green {
color: green;
}
.bold {
font-weight: bold;
}
Chaining <h1 class='special'>Text</h1> h1.special {
color: green;
}
Descendents <ul class='main-list'>
<li>Text</li>
</ul> 		.main-list li {
color: green;
}
Multiple Selectors h1, .menu {
color: green;
}

Specificity: ID > Class > Type

Note: Adding more than one tag, class, or ID increases the specificity.
